# Project Memory

## Core
Conference toolkit app for fundraising professionals. Mobile-first, no AI generation.
The AI Dude brand (primary) + Festival del Fundraising (secondary accent only).
Space Grotesk headings, DM Sans body. Deep indigo primary (#5b4de3), cool gray bg.
No gradients, no glassmorphism, no warm coral. Restrained editorial workbook feel.
Local storage for persistence. No login required. All content is structured worksheets/forms.
